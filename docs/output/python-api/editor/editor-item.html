Editor-Item¶

This class defines the graphical widget which allows for viewing and editing of models on a scrollable, zoomable canvas, with custom drawing styles available to each model.

class editor_item.EditorItemWidget(parent=None, ei=None)¶

Wrapper class for EditorItem so that an EditorItem can be configured and automatically used with EditorItemDelegate

class editor_item.EditorItemDelegate(parent=None)¶

Handles the mapping between EditorItem‘s data and the model’s data. Ensures that whenever the item’s data are edited, the model’s data get updated and vise-versa. This eables ensuring that the graphics object’s text label changes whenever its associated model changes, for instance.

class editor_item.EditorItem(index, parent=None)¶

Graphical representation of the data-model.
