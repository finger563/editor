Graphics-Items¶

These classes are custom items implementing draw styles not natively found in qgraphics*items

class graphics_items.TextItem(text='', parent=None, scene=None, ha=1, va=32)¶

Acts as a label for a EditorItem. Extends QGraphicsTextItem with more functionality for setting the alignment and position of the text relative to its parent EditorItem.

class graphics_items.RoundRectItem(x, y, w, h, xr=0.1, yr=0.1, parent=None)¶

Extends QGraphicsRectItem to support drawing a rectangle with rounded edges.
