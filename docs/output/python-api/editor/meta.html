class meta.Attribute(kind, value)¶

Generic Attributes class

Each Attribute has the following: kind – The datatype of the attribute e.g. string, pointer, float, bool etc. value – The value of the attributes e.g. ‘my_component’, 1.642 etc.

class meta.Model(parent=None)¶

Generic Model/Container class

Every Model has the following: parent – A parent Model Object. children – A list of children (Model) objects. attributes – A dictionary of attributes.
