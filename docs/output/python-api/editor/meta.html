class meta.Attribute(kind, value)¶

Generic Attributes class

Each Attribute has the following:

  • kind – The datatype of the attribute e.g. pointer, float, bool etc.
  • value – The value of the attributes e.g. ‘my_component’, 1.642 etc.
class meta.Model(parent=None)¶

Generic Model/Container class

Every Model has the following:

  • parent – A parent Model Object.
  • children – A list of children (Model) objects.
  • attributes – A dictionary of attributes.

class meta.Children(it=(), allowed=(), cardinality=())¶

Children list which extends MutableSequence and enforces item type and cardinality rules.

_inner – Contents of the list _allowed – The list will accept only object types contained in _allowed _cardinality – Cardinality of each accepted type

Parameters:
  • allowed (in) – List containing the allowed types of children
  • cardinality (in) – Dictionary of key:value pairs mapping object type to its cardinality string, e.g. Model: ‘0..*’
