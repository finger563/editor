Attribute Editors¶

These classes allow for special datatypes to be edited and updated by the MVC paradigm interface with through a QItemDelegate subclass.

class attribute_editors.FileEditor(name, fname, file_type, parent=None)¶

Subclass of QtGui.QPushButton that enables MVC ItemDelegate to set and query the filename. When the button is clicked it spawns a QFileDialog for selecting the file based on the file_type specified.

class attribute_editors.ListEditor(*args)¶
class attribute_editors.ComboSortFilterProxyModel(*args)¶

Subclasses QtGui.QSortFilterProxyModel to provide a proxy model to a QtGui.QComboBox for selecting references to other objects. By setting the rootIndex of the model and customizing the filterAcceptsRow() function, the reference scope and data-type can be enforced.

class attribute_editors.ReferenceView(parent=None)¶

Required for ReferenceEditor, handles the view of tree models.

class attribute_editors.ReferenceEditor(parent=None)¶

Required so that we can change how comboboxes show trees.

class attribute_editors.CodeEditor(*args)¶

Multi-line text editor subclassing QtGui.QTextEdit to allow for the editing of ‘code’ objects, such as ROS Message definitions, python code, etc. Supports code completion and syntax highlighting.
