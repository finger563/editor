Item-Model¶

This implements the model for the metamodel in the MVC paradigm. It uses Qt’s AbstractItemModel base class.

class item_model.ItemModel(root, parent=None)¶

Implements the QAbstractItemModel to interact with the underlying data-model, and create QModelIndex objects for retrieving and setting data.

index(row, column, parent)¶

Returns a QModelIndex for the child of the parent at row, col.

Parameters:
  • int row (in) – row number
  • int column (in) – column number
  • parent (in) – QModelIndex of the parent
parent(index)¶

Returns a QModelIndex for the parent of index.

insertRows(position, rows, parent=<PyQt4.QtCore.QModelIndex object>, _type=None)¶

Adds children to the model at position.

Parameters:
  • position (in) – a QModelIndex where the new children go
  • rows (in) – number of new children to insert
  • parent (in) – QModelIndex of the parent of new children
removeRows(position, rows, parent=<PyQt4.QtCore.QModelIndex object>)¶

Removes children from the model at position.

Parameters:
  • position (in) – a QModelIndex where the children start
  • rows (in) – number of children to delete
  • parent (in) – QModelIndex of the parent of the children
class item_model.SortFilterProxyModel(parent)¶

Extends QSortFilterProxyModel to customize filtering on a QAbstractItemModel.
