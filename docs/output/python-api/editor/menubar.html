Menubar¶

MenuBar for the Meta-Model Editor

This program provides an API for library code to add menus and actions to the GUI menubar.

menubar.menubar_add_menu(self, menu_name)¶
Parameters:string menu_name (in) – name of the menu, shortcut (denoted by &) optional (note that the & will be stripped)
menubar.menu_add_action(self, menu_name, menu_action)¶
Parameters:
  • string menu_name (in) – name used to create the menu from Editor.menubar_add_menu().
  • Action menu_action (in) – action.Action object to bind to the menu.
