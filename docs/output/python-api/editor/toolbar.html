Toolbar¶

Toolbar for the Meta-Model Editor

This program provides an API for library code to add toolbars, buttons and actions to the GUI.

toolbar.toolbar_create(self, tb_name)¶
Parameters:string tb_name (in) – name of the toolbar
toolbar.toolbar_add_action(self, tb_name, tb_action)¶
Parameters:
  • string tb_name (in) – name used to create the toolbar from Editor.toolbar_create().
  • Action tb_action (in) – action.Action object to bind to the toolbar.
toolbar.toolbar_add_widget(self, tb_name, tb_widget)¶
Parameters:
  • string tb_name (in) – name used to create the toolbar from Editor.toolbar_create().
  • Action tb_widget (in) – any pyQt widget to add to the toolbar.
